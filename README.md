# 🎬 Netflix Data Analysis Dashboard – Day 16 of 30-Day Coding Challenge

This is **Day 16** of my **30-Day Coding Challenge with Swara Gharat**, marking **Week 3 – Data Science & Visualization**.  
We built a **modern GUI dashboard** to perform **Exploratory Data Analysis (EDA)** on the Netflix dataset from Kaggle.

---

## ⚡ Features
- 📂 Load Netflix dataset from CSV  
- 📊 Interactive tabs to explore:
  - Top Genres  
  - Ratings Distribution  
  - Content Release Trend by Year  
  - WordCloud of Descriptions  
- 🎨 Modern dark theme GUI with **ttkbootstrap**  
- ☁ Auto-generated WordCloud from Netflix descriptions  
- 🖥️ Smooth, tabbed interface for easy navigation  

---

## 💡 What We Learned
- Cleaning and preprocessing real-world datasets  
- Using **Pandas + Seaborn + Matplotlib** for data visualization  
- Generating **WordClouds** for text data analysis  
- Building **modern GUIs** with ttkbootstrap & Tkinter notebooks  
- Combining Data Science + GUI programming for interactive dashboards  

---

- Python
